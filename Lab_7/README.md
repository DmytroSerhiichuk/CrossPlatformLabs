Лабораторна робота №7
===

Для запуску тестів виконати одну з команд:
* PostgreSQL:
  ```bash
  npm test -- --DbType=PostgreSQL
  ```
* SQLServer:
  ```bash
  npm test -- --DbType=PostgreSQL
  ```
* SQLite:
  ```bash
  npm test -- --DbType=SQLite
  ```
* InMemory:
  ```bash
  npm test -- --DbType=InMemory
  ```
